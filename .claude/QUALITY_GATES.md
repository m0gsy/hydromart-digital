# QUALITY GATES

Every milestone MUST satisfy all quality gates before it is considered complete.

======================================================
ENGINEERING
======================================================

✓ Project builds successfully

✓ Lint passes with zero errors

✓ Type checking passes

✓ Unit tests pass

✓ Integration tests pass

✓ End-to-End tests pass (when applicable)

✓ No breaking API changes unless documented

✓ Database migrations validated

✓ Documentation updated

======================================================
ARCHITECTURE
======================================================

✓ Clean Architecture preserved

✓ SOLID principles maintained

✓ DDD boundaries respected

✓ No unnecessary coupling

✓ No duplicated business logic

✓ Repository pattern respected

✓ Dependency Injection preserved

======================================================
SECURITY
======================================================

✓ Security review completed

✓ OWASP Top 10 considered

✓ Input validation complete

✓ Authorization verified

✓ Authentication verified

✓ Secrets properly managed

✓ Audit logging implemented where required

======================================================
PERFORMANCE
======================================================

✓ No unnecessary database queries

✓ N+1 queries avoided

✓ Pagination implemented

✓ Caching strategy reviewed

✓ Background jobs used where appropriate

✓ Lighthouse target met (Frontend)

======================================================
UI / UX
======================================================

✓ Taste Skill review completed

✓ Responsive design verified

✓ Accessibility verified

✓ Loading state implemented

✓ Empty state implemented

✓ Error state implemented

✓ Success state implemented

✓ Skeleton implemented where appropriate

======================================================

======================================================

✓ Humanizer review completed

✓ Marketing Skills applied where customer-facing communication exists

✓ Copy reviewed for clarity and professionalism

======================================================

A milestone is NOT complete until every applicable quality gate has passed.

======================================================

✓ Business requirement satisfied

✓ Acceptance criteria satisfied

✓ Backend completed

✓ Frontend completed

✓ Mobile completed (if applicable)

✓ Database completed

✓ API documented

✓ Tests passed

✓ Security reviewed

✓ Performance reviewed

✓ Documentation completed

✓ Claude Memory updated
