# DESIGN PRINCIPLES

Every UI must be

- Premium
- Modern
- Minimal
- Mobile First
- Accessible
- Consistent
- Fast

Always prefer

- Reusable Components
- Design Tokens
- Clear Typography
- Strong Visual Hierarchy
- White Space
- Progressive Disclosure

Never

- Generic Dashboard UI
- Bootstrap Look
- Inconsistent Components
- Duplicate Patterns
- Random Colors
- Random Spacing
- Hardcoded Styles
